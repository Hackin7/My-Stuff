Not So Silent

Problem

Can you figure out the admin's password from this packet capture?

Given: noisy.pcap

Solution

Check up https://developer.mozilla.org/en-US/docs/Web/HTTP/Authentication

You notice packet 4241 has a HTTP OK Response, so the GET request before that has the password. which is 4239 * filter with http.response.code==200
